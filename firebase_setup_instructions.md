Agar aap ko console mein "Permission blocked" ka error aa raha hai, toh iska matlab hai ke aap ke browser ne notifications ko mana kar diya hai. 

### Isay theek kaise karein?
1. **URL bar mein Lock icon ðŸ”’**: Browser ki address bar mein jo lock icon hota hai, us par click karein.
2. **Notifications Toggle**: Wahan 'Notifications' ka button hoga, usay **Reset** karein ya **Allow** par set karein.
3. **Refresh**: Page ko refresh karein.
4. **Allow Popup**: Jab browser dobara permission mangay, toh "Allow" par click karein.

### Note:
Browsers aksar tab notifications block karte hain jab unhein user interaction (click) ke bagair manga jaye. Maine code ko behtar kar diya hai taake ye gracefully handle ho, lekin pehli baar aap ko manually browser ki settings se unblock karna par sakta hai.
